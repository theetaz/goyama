# @goyama/web-client

Farmer-facing SPA. Vite + React + TypeScript + TanStack Router + TanStack Query + Shadcn UI + Tailwind.

## Dev

```bash
# from repo root
pnpm install
pnpm --filter @goyama/web-client dev        # serves on :5173

# in a second terminal
cd services/api && make run                   # serves on :8080
```

The Vite dev server proxies `/v1/*` to the Go API so the SPA can fetch without CORS hiccups.

## Layout

```
src/
├── main.tsx              # Root entry + TanStack Query/Router providers
├── routes/
│   ├── __root.tsx        # App shell with header, nav, locale picker
│   ├── index.tsx         # Home / crop explorer (search + category filters)
│   └── crops.$slug.tsx   # Crop detail page
├── components/ui/        # Shadcn components (button etc)
├── lib/
│   ├── api.ts            # Typed API client
│   └── utils.ts          # cn(...) helper
├── styles/globals.css    # Tailwind + design tokens
└── i18n/
    ├── index.ts          # i18next bootstrap
    └── locales/          # en, si, ta JSON
```

Route tree is autogenerated to `src/routeTree.gen.ts` by `@tanstack/router-plugin` on dev start.
